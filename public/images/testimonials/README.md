# Testimonial Images

## Purpose
Profile photos for client testimonials and reviews.

## Specifications
- **Dimensions:** 128x128px minimum (can be larger, will be scaled down)
- **Aspect Ratio:** 1:1 (square)
- **Format:** JPG or WebP
- **File Size:** Max 50KB
- **Color Space:** sRGB
- **Display:** Will be rendered as circular avatars

## Content Guidelines
- Professional headshots or portraits
- Good lighting, clear focus
- Smiling, friendly expressions
- Plain or simple backgrounds preferred
- Face should be clearly visible
- Ensure subjects are centered (circular crop consideration)

## Naming Convention
- `testimonial-{name}-{id}.{ext}`
- Examples:
  - `testimonial-john-smith-1.jpg`
  - `testimonial-mary-jones-2.jpg`

## Privacy Considerations
- Obtain proper consent for using photos
- Consider using stock photos or illustrations if needed
- Ensure HIPAA compliance for any real patient images

## Alternative Options
- Use initials with colored backgrounds
- Generic avatar illustrations
- Silhouette images for privacy
